# Fonts

The configuration uses `segoeuithibd.ttf` (Segoe UI Thin Bold / Condensed).

**This font is proprietary (Microsoft) and cannot be included in the repository.**

## How to obtain it

- **Windows**: copy from `C:\Windows\Fonts\segoeuithibd.ttf`
- **macOS / Linux**: extract from a Windows installation or Office package

## Open-source alternative

[Noto Sans](https://fonts.google.com/noto/specimen/Noto+Sans) is a visually
similar, freely licensed substitute.  Download `NotoSans-Thin.ttf` and update
the `font:` section in `coreink.yaml` accordingly.
